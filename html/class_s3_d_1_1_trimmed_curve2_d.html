static ResultStatusEnum S3D::TrimmedCurve2D::Create ( Curve2D * basisCurve,

TrimmedCurve2DResult & trimmedCurveResult )
static

Create a trimmed curve from a basis curve. If trim0 < trim1, then the direction of the trimmed curve is the same as the direction of the basis curve. If trim0 > trim1, then the direction of the trimmed curve is the opposite to the direction of the basis curve.

Parameters
[in]basisCurveThe basis curve. The basis curve must not be a trimmed curve.
[in]trim0The start parameter of the trimmed B-Spline 2D curve.
[in]trim1The end parameter of the trimmed B-Spline 2D curve.
[out]trimmedCurveThe trimmed curve.
Returns
RS_Success
RS_Bad_Parameter
RS_Bad_Curves
