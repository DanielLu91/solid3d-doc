Join a list of B-Spline 2D curves.

Parameters
[in]bsplineListThe list of B-Spline 3D curves. The curves must be in the correct order. The end of previous curve must coincide with the start of the next curve.
[in]jointKnotsThe ascending knot values at joint, the size must be equal with 0 or nBsplineList + 1. If the size is 0, the domain of resulting bspline will be [0, 1].
[out]bsplineThe B-Spline 2D curve.
Returns

Trim the B-Spline 2D curve. If trim0 < trim1, then the direction of the trimmed B-Spline 2D curve is the same as the direction of the basis curve. If trim0 > trim1, then the direction of the trimmed B-Spline 2D curve is the opposite to the direction of the basis curve.

Parameters
[in]t0The start parameter of the trimmed B-Spline 2D curve.
[in]t1The end parameter of the trimmed B-Spline 2D curve.
[out]trimmedBSplineThe trimmed B-Spline 2D curve.
