Create a B-Spline 3D curve by fitting. This function generates a B-Spline 3D curve that approximates the given set of points using a least-squares fitting approach. The number of control points can be specified independently from the number of data points.

Parameters
[in]degreeThe degree of the B-Spline 3D curve.
[in]nbControlPtsThe number of control points for the B-Spline curve.
[in]pointsSet of 3D data points to be approximated.
[in]pointWeightsWeights for each data point in the fitting process.(default: all 1.0) If pointWeights[i] > 0.0, it is an unconstrained point; otherwise, it is a constrained point (requiring exact interpolation).
[in]derivativesDerivative value array(default: empty)
[in]derivativeIndicesIndexes of derivative value array(default: empty)
[in]derivativeWeightsWeights of derivative value array(default: empty) If derivativeWeights[k] > 0.0, it is an unconstrained derivative; otherwise, it is a constrained derivative (must be exactly satisfied).
[out]bsplineThe resulting B-Spline 3D curve.
Returns
RS_Success
RS_Bad_Degree
RS_Insufficient_Points
RS_Invalid_Weight
RS_Bad_Knots
RS_Periodic_Open
RS_Periodic_Not_Smooth
