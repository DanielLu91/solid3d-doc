Create a B-Spline surface by Interpolation.

Parameters
[in]gridThe grid of points to interpolate. --------------> v Direction grid[0][0] -— grid[0][1] -— grid[0][2] ... | | | grid[1][0] -— grid[1][1] -— grid[1][2] ... | | | grid[2][0] -— grid[2][1] -— grid[2][2] ...
[in]uDegreeThe degree of the B-Spline surface in U direction.
[in]vDegreeThe degree of the B-Spline surface in V direction.
[in]uParamsThe parameters in U direction for the grid points. If empty, will be computed automatically. The size must be equal to grid.size() or 0. If 0, will be computed automatically.
[in]uFlatKnotsThe flattened knot vector in U direction of the resulting surface. If empty, will be computed automatically. The size must be equal to grid.size() + uDegree + 1 or 0. If 0, will be computed automatically.
[in]vParamsThe parameters in V direction for the grid points. If empty, will be computed automatically. The size must be equal to grid[0].size() or
  1. If 0, will be computed automatically.
[in]vFlatKnotsThe flattened knot vector in V direction of the resulting surface. If empty, will be computed automatically. The size must be equal to grid[0].size() + vDegree + 1 or 0. If 0, will be computed automatically
[out]bsplineSurfaceThe resulting B-Spline surface.

static ResultStatusEnum S3D::BSplineSurface::CreateNetSurface ( const std::vector< BSpline3D * > & uCurves,
const std::vector< BSpline3D * > & vCurves,
const double intersectionTol,
BSplineSurface & bsplineSurface )
static

Create a Gordon surface from two sets of B-Spline curves and their intersection points.

  • All input curves must be NURBS or NURBS-like. only non-rational curves are supported i.e., without weights.
  • All U-curves must be have “the same" direction; all V-curves must be also have "the same" direction.
  • U-curves must be ordered along direction of V-curves, and vice versa.
    Parameters
    [in]uCurvesThe B-Spline curves along U direction.
    [in]vCurvesThe B-Spline curves along V direction.
    [in]intersectionTolThe tolerance for intersection points. Default = Precision::LengthTol
    [out]bsplineSurfaceThe resulting B-Spline surface.
    Returns
    RS_Success
    RS_Insufficient_Curves
    RS_Curves_Dont_Intersect
    RS_Bad_Curves
