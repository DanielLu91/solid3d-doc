Create a trimmed curve from a basis curve. The basis curve does not holds any attributes. The trimmed curve also does not hold attributes. If trim0 < trim1, then the direction of the trimmed curve is the same as the direction of the basis curve. If trim0 > trim1, then the direction of the trimmed curve is the opposite to the direction of the basis curve.

Parameters
[in]basisCurveThe basis curve.
[in]trim0The start parameter of the trimmed B-Spline 2D curve.
[in]trim1The end parameter of the trimmed B-Spline 2D curve.
[out]trimmedCurveThe trimmed curve.
